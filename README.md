# You've heard it, but have you heard of it?

## Why should you care?

1) You don't have to care. It's pretty cool though. I think you should.

2) It's influenced/created quite a few genres too:
    - *Dub*step
    - Dancehall
    - Drum and bass
    - Hip hop
    - Ska
